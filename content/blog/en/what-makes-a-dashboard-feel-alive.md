# What Makes a Page Feel Alive

A page doesn't feel alive because it uses a lot of animation. It feels alive because it still has subtle changes when your attention shifts, reminding you that something is happening.

I made this site's homepage into a space with floating glows and layered cards. It doesn't need to forcefully tell you "this is flashy." It wants you to feel a gentle, ongoing rhythm as you move your mouse, pause, and look around.

## Where liveliness comes from

- subtle motion that keeps the page from feeling stiff
- clear hierarchy so information remains orderly
- enough whitespace so the eyes can rest
- a unified theme so every section feels like a branch of the same tree

This is the state I most want to preserve for the blog: the page has rhythm, content has weight, and emotion has room.
